# RPG game

## 预计使用的工具

- Visual Studio 17 2022 (桌面C++开发，默认配置)
- easyX
- C++ 11
- engine of silk song

## 开发注意事项

1. 每个类要考虑是否实现六个函数/方法：构造、拷贝构造、转移构造、赋值、转移赋值、析构。如果要去掉其中一种方法，在方法说明的尾部加上`=delete`。

2. 确认不抛出异常的函数/方法，在说明的尾部加上`noexcept`修饰符。析构函数一定不能抛出异常，所以都加上`noexcept`。

3. 公有成员放前面，保护成员其次，私有成员放最后。

4. 使用MVFM模式开发（lec2 Page122）。
